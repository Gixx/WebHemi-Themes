This website uses such modern technologies for the user interface, that this browser does not support, so it would unable to display the content in the correct form.

To develop a compatible rendering would cost too much time, money and energy, that nobody will give me, since it's an open source software. So I decided to kick out this browser from the game.

But if you are still interested in my website, then I suggest you to download and install one of the following browsers. The list is not complete, I included only those I know to be compatible with my website. The order of the list follows my personal preferences.

List of the compatible browsers

Google Chrome

In the aspect of technology, this is one of the best browsers. It is available in most platforms, it is fast, easz to use and it has a clean user interface. It has issues of course: it loves when the PC is equipped with a huge amount of memory, but yet it has the newest features.

Download
Mozilla Firefox

The winner of the second great browser war, but its shine is a bit faded since then, because it became more and more bigger and slower. But now it seems they get themselves together and go to the next battle with a brand new browser. Whoever loves it, will use it.

Download
Opera

Always this browser was the "small" and "alternative" on the market. Unfortunately the markating strategy failure (banners were placed in the free version) made a bad mark on its reputation, and that's why it was not that popular world wide. But those times have gone, and this browser is renewed too, and tries to catch some wind out of the Chrome's sails.

Download
Apple Safari

The inevitable and default browser in the Apple Macintosh systems. For a short time it was also released to Microsoft Windows, but besides the Chrome and the Firefox it had no change to grow, so it was not worth to invest time and money in the Windows port any more.

Microsoft Edge

After the Internet Explorer completely became the target of scorn and jokes among the developers and maybe also among the end users, the Microsoft decided to restart from scratch, and build and release a brand new browser that meets the challenges of today's age. It's the part of ever Windows 10 system, but for earlier versions it's not available officially.
